Frequently Asked Questions

Synthetic data is a type of artificially generated data, made by stadistic and machine learning algorithms. It's principally used for solving database-privacy problems when there is a need on training models with data that is private or sensitive.

Sometimes, using real data is not possible due to its expensive collect process or when that data is private or sensitive. Synthetic data can fix these problems genereting data that does not belong to anyone and can be used to create more diverse and balanced datasets.

Synthetic data is generated using stadistic and machine learning algorithms. These algorithms are trained with real data and then they generate new data that retains the structure of the original data but is not the same.

With Synthetic Data Generator you can generate synthetic data from your own data. You can upload your data using a csv format, select the columns you want to generate synthetic data from and select the synthesizer (the algorithm) that you want to use. Then, you can download the synthetic data generated.

First, you need to register and login. Then, go to the generate section and follow the instructions.

Synthetic Data Generator is a web application made with Flask (a Python web framework) and Bootstrap (a CSS framework). The synthetic data is generated using SDV (Synthetic Data Vault), a Python library that provides a set of tools to generate synthetic data.
