About the project

This project was made by Francisco Javier Molina Rojas as his Bachelor´s Degree Final Project, and it was directed by Dr. Juan Manuel Dodero Beardo and codirected by Dra. Mercedes Rodríguez García.

The project uses:

  • Python as the main programming language.
  • Flask as the web framework.
  • Bootstrap as the CSS framework.
  • SQLite as the database.
  • SDV (Synthetic Data Vault) as the library that provides a set of tools to generate and evaluate synthetic data.
